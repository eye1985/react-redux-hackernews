#react-redux-hackernews

A hacker-news webapp with an option to refresh every 30 second.
The main purpose for this app is to get familiar with React/Redux.

##Build with
- Babel 6
- ES6
- React
- React-redux
- Webpack
- Material CSS

##How to run

`npm i`  
`npm start`

[Demo](https://vivid-heat-6972.firebaseapp.com/)
